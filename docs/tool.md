## 🎓 입문 단계 (아두이노)

### 필수 하드웨어

#### 1. 아두이노 보드

| 보드 | 가격 | 특징 | 추천 용도 |
|------|------|------|----------|
| **Arduino Uno R3** | ~₩15,000 | 가장 기본적인 보드 | 학습용, 기본 프로젝트 |
| **Arduino Nano** | ~₩8,000 | 작고 저렴함 | 소형 프로젝트 |
| **Arduino Mega 2560** | ~₩25,000 | 많은 핀, 큰 메모리 | 복잡한 프로젝트 |
| **Arduino Leonardo** | ~₩18,000 | USB HID 지원 | 키보드/마우스 에뮬레이션 |

**추천**: Arduino Uno R3 (가장 많은 자료와 커뮤니티 지원)

#### 2. 아두이노 스타터 키트

**Arduino Starter Kit (공식)**
- 가격: ~₩80,000
- 포함 내용:
  - Arduino Uno R3
  - 브레드보드
  - 점퍼 와이어
  - LED (다양한 색상)
  - 저항 (다양한 값)
  - 버튼, 스위치
  - 포텐셔미터
  - 센서 (온도, 빛, 기울기)
  - LCD 디스플레이
  - 서보 모터
  - DC 모터
  - 프로젝트 북

**대안: 호환 키트**
- 가격: ~₩30,000-50,000
- 중국산 호환 보드 + 센서 모듈 세트
- 가성비 좋음 (학습용으로 충분)

#### 3. 기본 부품

| 부품 | 수량 | 가격 | 용도 |

### 구매 팁

1. **시작은 키트로**
   - 스타터 키트 구매 추천
   - 개별 구매보다 저렴
   - 필요한 부품 대부분 포함

2. **호환 보드 활용**
   - 정품 대비 1/3-1/2 가격
   - 학습용으로 충분
   - 대량 구매 시 유리

3. **해외 직구**
   - 국내 대비 50-70% 저렴
   - 배송 시간 고려 (2-4주)
   - 소액 구매 시 관세 없음

4. **중고 거래**
   - 당근마켓, 중고나라
   - 측정 장비는 중고도 OK
   - 상태 확인 필수
